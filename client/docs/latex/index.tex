Welcome to the documentation for the SOSE25 Game Client, a C++ Open\+GL-\/based game engine designed for space simulation and exploration games. This client features a modern rendering pipeline with both forward and deferred rendering capabilities.\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Key Features}{Key Features}}\label{index_features_sec}

\begin{DoxyItemize}
\item {\bfseries{Modern Open\+GL Rendering}}\+: Supports both forward and deferred rendering pipelines
\item {\bfseries{\doxylink{class_shader}{Shader} Management}}\+: Comprehensive shader system for various rendering techniques
\item {\bfseries{Cross-\/\+Platform Support}}\+: Builds on mac\+OS, Windows, and Linux
\item {\bfseries{Web\+Socket Communication}}\+: Real-\/time communication with game servers
\item {\bfseries{Asset Management}}\+: Efficient loading and management of 3D models, textures, and fonts
\item {\bfseries{\doxylink{class_u_i}{UI} System}}\+: Integrated user interface rendering system
\item {\bfseries{\doxylink{class_input}{Input} Handling}}\+: \doxylink{struct_keyboard}{Keyboard} and mouse input management
\item {\bfseries{Profiling Tools}}\+: Built-\/in performance profiling and debugging utilities
\end{DoxyItemize}\hypertarget{index_structure_sec}{}\doxysection{\texorpdfstring{Project Structure}{Project Structure}}\label{index_structure_sec}
