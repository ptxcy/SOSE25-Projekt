Diese Anleitung erklärt, wie du die Entwicklungsumgebung für den C++ Client mit Crow einrichtest.\hypertarget{md_doc_2crow_autotoc_md14}{}\doxysection{\texorpdfstring{Voraussetzungen}{Voraussetzungen}}\label{md_doc_2crow_autotoc_md14}

\begin{DoxyEnumerate}
\item {\bfseries{C++ Compiler\+:}} Ein C++17-\/kompatibler Compiler wird benötigt. Das Build-\/\+Skript verwendet {\ttfamily clang++}. Unter mac\+OS ist dieser normalerweise mit den Xcode Command Line Tools enthalten.
\item {\bfseries{Open\+SSL\+:}} Wird für HTTPS-\/\+Verbindungen benötigt.
\begin{DoxyItemize}
\item {\bfseries{mac\+OS\+:}} {\ttfamily brew install openssl}
\item {\bfseries{Ubuntu/\+Debian\+:}} {\ttfamily sudo apt-\/get install libssl-\/dev}
\end{DoxyItemize}
\item {\bfseries{Bear\+:}} Wird für die Generierung von {\ttfamily compile\+\_\+commands.\+json} verwendet (für bessere IDE-\/\+Integration).
\begin{DoxyItemize}
\item {\bfseries{mac\+OS\+:}} {\ttfamily brew install bear}
\item {\bfseries{Ubuntu/\+Debian\+:}} {\ttfamily sudo apt-\/get install bear}
\end{DoxyItemize}
\item {\bfseries{Unzip\+:}} Wird für die Installation von Crow benötigt.
\begin{DoxyItemize}
\item {\bfseries{mac\+OS\+:}} Normalerweise bereits installiert
\item {\bfseries{Ubuntu/\+Debian\+:}} {\ttfamily sudo apt-\/get install unzip}
\end{DoxyItemize}
